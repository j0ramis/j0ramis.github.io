About

This page is built in part to demonstrate work for the Charlotte class ITIS3135 - Front-End Web Application Development, for sections run by Mr. von Briesen. You may find additional information about this course at D.I. von Briesen's Website.

Throughout this course, I will be applying my concentration in Human-Computer Interaction to focus on creating websites that are not only structurally sound and valid but also highly usable, accessible, and visually clean.

Tools Used

  • VS Code: My primary code editor used to write, inspect, and organize HTML, CSS, and JS files.
  • Emmet: Built-in toolkit for writing HTML shorthand structures and saving development time.
  • FileZilla: An open-source FTP client used to transfer files securely to the server directory.
  • Git & GitHub Desktop: Version control tools utilized to track and push structural commits cleanly.
  • Browser DevTools: Utility inspectors used to troubleshoot and debug responsive interface styling in real-time.
